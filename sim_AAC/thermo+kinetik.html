Simulation B: Hess'sches Gesetz

Überblick der Simulation:

  • Berühren, halten und ziehen Sie, um Atome zu verschieben.
  • Tippen Sie auf eine Bindung, oder tippen Sie nacheinander auf zwei Atome, um eine Bindung zu brechen (✄).
  • Tippen Sie nacheinander auf zwei Atome, um eine Einzel- (●⚊●) oder Doppelbindung (●⚌●) zu bilden.
  • Die Enthalpieänderungen werden in der Grafik unten angezeigt. Sie können auch alle Bindungen auf einmal aufbrechen (●✕●✕●).
  • Wenn Sie auf "Zurücksetzen" klicken, wird der Pfad von Methan- und Sauerstoffmolekülen neu gestartet.

Beachte:

  • Jedes Molekül in der Simulation entspricht 1 mol von gasformigen Molekülen.
  • Tippen Sie auf eine Bindung, oder tippen Sie nacheinander auf zwei Atome, um eine Bindung zu brechen oder zu bilden.

Hilfreiche Informationen

Konstanten:

  • Verdampfungsenthalpie von Wasser: ΔHvap = 43,9 kJ mol-1
  • Sublimationsenthalpie von Kohlenstoff: ΔHsub = 717 kJ mol-1
  • Gaskonstante: R = 8,314 J mol-1 K-1
  • Coulomb-Konstante: k = 8,99·109 N m2 C-2

    • Gleichungen:

  • Coulombsches Gesetz: F = k (q1q2)/r2
  • Gibbs-Energie: ΔGo = ΔHo - T ·ΔSo=-RT·ln(Keq)
